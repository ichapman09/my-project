Data Science Lifecycle Tutorial by Isaiah Chapman

Introduction

The National Football League is the most watched sports league in the United States of America. The league consists of 32 teams across the country. Each team plays 17 games every season, but before the 2021 season, every team played 16 games. The goal of each team is to win as many games as possible, make it to the postseason, and win the Super Bowl.

result_df = pd.DataFrame()

for index, row in temp_df.iterrows():
    # Create two new rows with alternating values from the original row
    away = {}
    home = {}

    # Looping through each columns
    for i, col in enumerate(temp_df.columns):
        # The first three rows should be the same for both the away and home team
        if i < 3:
            away[col] = row[col]
            home[col] = row[col]
        # If the column name end is away it is a stat for the away team else home team
        elif col.endswith('away'):
            away[col[:-5]] = row[col]
        else:
            home[col[:-5]] = row[col]
            
    # Changing column to team_name
    away['team_name'] = away.pop('')
    home['team_name'] = home.pop('')

    # Finding the winner for the game and giving that team one win
    if away['score'] > home['score']:
        away['wins'] = 1
        home['wins'] = 0
    elif away['score'] < home['score']:
        away['wins'] = 0
        home['wins'] = 1
    else:
        away['wins'] = 0
        home['wins'] = 0

    # Append new rows to the result DataFrame
    result_df = result_df._append(away, ignore_index=True)
    result_df = result_df._append(home, ignore_index=True)

def made_playoffs(reg_row, post_df):
    # Get the team and season
    season = reg_row.get('season')
    team = reg_row.get('team_name')

    # Searching post season data to see if that team was in the post season that season
    matching = post_df[(post_df['season'] == season) & (post_df['team_name'] == team)]
    
    return 1 if not matching.empty else 0

def grad_descent_log_regression(X, y, num_iterations, learn_rate):
    m, n = X.shape
    theta = np.zeros(n)
    bias =  0

    # Looping through all iterations
    for i in range(num_iterations):

        # Hypothesis function
        z = np.dot(X, theta) + bias
        y_pred = 1/(1 + np.exp(-z))

        # Optimization
        dw = (1 / m) * np.dot(X.T, y_pred - y)
        db = (1 / m) * np.sum(y_pred - y)

        # Changing the values of the weights and biases
        theta -= learn_rate * dw
        bias -= learn_rate * db
        
    return theta, bias

Before we put our data into the model we must first normalize the data. This is due to each field having different ranges of values. Having a huge range of values can cause the model to have a heavier weight towards a certain field when it should not.

Also with this model, you have a learning rate and a number of iterations. The learning rate controls how much the weights can be changed each iteration. A small learning rate can cause your algorithm to not reach the minimum point. A large learning rate can cause your algorithm to diverge. The number of iterations is how many times the algorithm runs before finding the minimum point.

Insight & Policy

This is the last part of the data science lifecycle. Here is where the model outcome is talked about and what can be done using the model.

From our model, we can see that wins have the most impact on predicting whether a team makes the postseason with passing yards next. Also, you can see that fumbles have the least impact on whether a team makes the postseason out of all the fields in the model

We now have a working model that can use pass yards, rush yards, sack numbers, penalty yards, fumbles, interceptions, score, and wins to predict whether a team is likely to make it to the postseason. It is important to note that these are not the only variables that contribute to a team making the postseason. There are other variables that we dropped earlier, variables not included, and variables that do not fit traditionally into data. A team can have high offensive stats, but their defense can give up a lot of points which causes them to lose games. Also, there could be a tie of two teams that could make it to the postseason and the NFL tiebreaker rules will determine who makes it or not.

A model is great for making predictions, but it should not be looked at as the absolute future.
